# PBAutoJoin plugin
# Plugin for B3 (www.bigbrotherbot.net)
# www.ptitbigorneau.fr

pbautojoin B3 plugin (v1.4) for Urban Terror

Requirements
------------

* BigBortherBot(3) >= version 1.10

Installation
------------

1. Copy the 'pbautojoin' folder into 'b3/extplugins' and 'pbautojoin.ini' file into '/b3/extplugins/conf'.

2. Open your B3.ini or b3.xml file (default in b3/conf) and add the next line in the [plugins] section of the file:
    for b3.xml
        <plugin name="pbautojoin" config="@b3/extplugins/conf/pbautojoin.ini"/>
    for b3.ini
        pbautojoin: @b3/extplugins/conf/pbautojoin.ini
